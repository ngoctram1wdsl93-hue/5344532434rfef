## Ребренд Phase 2.1 (completed)
### Позиціонування
Змінено з “помпи та пневмокомплекти” на **пневмоподушки / пневмопідвіска для комерційного транспорту**.

### Категорії (6)
- `air-springs` — Пневмоподушки
- `air-suspension` — Комплекти пневмопідвіски
- `components` — Комплектуючі (компресори, фітинги, магістралі, клапани)
- `installation` — Встановлення
- `custom-solutions` — Індивідуальні рішення
- `by-vehicle` — За авто (Sprinter/Crafter/Transit/вантажні/причепи)

### Товари
13 SKU, всі замінені: пневмоподушки Sprinter/Transit/Crafter, комплекти для вантажних і причепів, компресор, фітинги, манометр, електроблок, RV-комплект.

### FAQ
8 питань, переписані під пневмопідвіску (гарантія 12/24 міс, власне виробництво, підбір під авто, самостійна установка).

### Дизайн-система (Monochrome Industrial Premium)
- bg: `#F7F7F5`, surface: `#FFFFFF`, muted: `#F1F1EF`
- text: `#111111` / `#666666`, border: `#E7E7E7`
- CTA primary: чорна `#111111` → hover `#2A2A2A`
- CTA secondary: біла + border
- Повністю прибрано червоний `#EF4444` із UI
- Типографіка: Space Grotesk для заголовків (tight tracking), display-heading для hero (76px)
- Радіуси: 24–28px картки, 14–16px кнопки
- Filter: `img-neutral` (знижена сатурація) для всіх фото

### Секції (перебудовані)
- **Hero** — editorial 76px heading, центральне візуальне фото з floating spec card + stats strip
- **Categories** — 6 преміальних карток, gradient overlay, icon badge, product count
- **FeaturedProducts** — білий блок, продуктові картки з сумісністю + кнопка “Запитати ціну”
- **UseCases** — Mercedes Sprinter / VW Crafter / Ford Transit / вантажні / причепи / спецтехніка
- **WhyChooseUs** — 6 причин із нумерацією 01–06
- **HowItWorks** — темна `#111111` секція з 4 кроками (драматургія)
- **Gallery** — masonry-like grid з caption chips
- **FAQ** — accordion на мутному бекграунді
- **FinalCTA** — темний блок + вбудована форма підбору
- **Footer** — темний `#111111`, 4-колоночний, eyebrow в uppercase
- **Header** — sticky, біла логотип-плашка, CTA `Отримати підбір →`
- **FloatingContact** — чорна кнопка, меню з білими pills

## Phase 3.3 — Конверсійний апгрейд (completed)

### Ціль
Перехід від «сайт норм» → «залишають заявку без роздумів». Точкові правки по фідбеку: дизайн + копірайт + доміна́нта.

### HERO — переписаний під удар
- **Фон:** `hero-bg` = `linear-gradient(180deg, #FFFFFF 0%, #F5F5F3 55%, #F1F1EF 100%)` замість flat `#F7F7F5` → знято «серість»
- **Фото:** `img-neutral` піднятий `saturate(0.85)→0.95 + contrast(1.02)→1.05` → живіші без попси
- **Dark trust badge (новий):** чорний pill «10+ років виробництва пневмопідвіски в Україні» — ДОМІНАНТА у верхньому полі екрану
- **Другий тригер:** «Один із перших виробників пневмопідвіски в Україні» з зеленою пульсуючою крапкою
- **Subheadline:** переписаний → «Не просідає під навантаженням» (жирне, =біль клієнта) + «Підбираємо рішення… з урахуванням задачі, ваги та умов роботи» (експертність)
- **Vehicle chips:** префікс «Працюємо з» (uppercase 0.12em tracking) → перетворив просто список на експертний меседж
- **CTA:** «Отримати підбір під авто» → «Підібрати під авто» (коротше, яскравіше) + `cta-lift` тінь `0_10px_30px rgba(17,17,17,0.22)` → найтемніша точка на екрані
- **Microcopy під CTA:** ✓ «Безкоштовний підбір під ваше авто за 15 хвилин»
- **Trust strip (4 точки, переписаний):** `1000+ встановлених комплектів` · `10+ років досвіду` · `Власне виробництво` · `~1 год середня відповідь` — прибрано слабке «вся Україна»
- **Відступи:** `pt-10 → pt-12/16/24`, `pb-14 → pb-20/24/32`, `gap-10/14 → gap-12/16`
